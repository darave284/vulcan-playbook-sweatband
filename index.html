How It Works
GitHub Actions Pipeline
Three Python scripts run at 8am UTC daily. They sync Google Ads data into Asana automatically.

sync_changelog.py

Queries Google Ads API for all Sweatband account changes in the past 24 hours. Creates tasks in the Change Log Asana section. High-priority changes fire a Slack notification.

sync_alerts_to_asana.py

Reads unprocessed rows from the Alert Hub sheet. Maps severity → Asana priority. Creates tasks in Alerts section.

generate_recurring_tasks.py

Creates daily (6), weekly (19), and monthly (22) checklist tasks in Asana. Deduplicates against existing incomplete tasks.

Rules

1. Daily Checks before optimisation. Always.
2. Log every change. If you changed it, Change Log within 24 hours.
3. HIGH alerts = same day action. No exceptions.
4. Concerned/At Risk sentiment? Tell David immediately.
5. Come back after 7 days and fill in the Result field on Change Log entries.
6. Don't create tasks in Alerts. Automated only.
